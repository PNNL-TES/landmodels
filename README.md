# landmodels

This repository has been archived and is no longer maintained.
The code is provided for historical reference and may contain unpatched or unknown vulnerabilities.
It should not be used in production systems.

The published paper is:

R.: Leveraging observed soil heterotrophic respiration fluxes as a novel constraint on global-scale models, Glob. Chang. Biol., 27, 5392–5403, 2021. https://onlinelibrary.wiley.com/doi/10.1111/gcb.15795
